

# Full-Stack CRUD Application

---

## Project Description

This is a **Full-Stack CRUD Application** where users can securely **Sign Up, Login, and perform CRUD operations** on categories.

The application demonstrates complete frontend and backend integration using **React.js, Node.js, Express.js, and MongoDB**.

Authentication is implemented using **JWT Tokens**, and APIs are connected using **Axios**.

The project also includes protected routes, image upload functionality, and responsive UI design.

---

## Features

- **User Authentication:** Secure Signup and Login system.
- **JWT Authorization:** Token-based authentication for protected routes.
- **Create Category:** Users can add new categories.
- **Read Categories:** View all existing categories.
- **Update Category:** Edit category details.
- **Delete Category:** Remove unwanted categories.
- **Category Details:** View complete category information.
- **Image Upload:** Upload category images using Multer.
- **REST API Integration:** Frontend and backend communication using Axios.
- **Responsive UI:** Clean and user-friendly interface.

## How It Works

1. Users first create an account using the Signup page.
2. After successful signup, users log in securely.
3. JWT tokens are generated and stored in local storage.
4. Authenticated users can access the dashboard.
5. Users can perform Create, Read, Update, and Delete operations on categories.
6. Backend APIs store and manage data using MongoDB.
7. Frontend communicates with backend APIs using Axios.
